<h1 align="center">nyaBook</h1>

> 🪽 一个Web端的电话簿

## 介绍

这个作品作为两位贡献者的人机交互课程结课作业；虽然它不实用，也许算不上好用，但是一起合作开发完成作业，不是很酷吗？！

## 特性

由Spring+Mybatis plus & NodeJs+Vue3开发的响应式电话簿，可以建立个人主页、联系人列表、联系人群组列表，项目原型图`/design/view`，接口设计`/design/设计.md`。
